WDD 230 Presentation: JavaScript Event Listener

Welcome to my presentation on JavaScript Event Listener. I will be talking on the meaning, parameters, and the application of event listener.

MEANING

Event

Events are things that happen in the system you are programming, which the system tells you about so your code can react to them.

HTML Event

HTML events are "things" that happen to HTML elements.
When JavaScript is used in HTML pages, JavaScript can "react" on these events.
The use of JavaScript to react to event is called "Event Handler".

Examples:

An HTML event can be something the browser does, or something a user does.
Here are some examples of HTML events:

An HTML web page has finished loading
An HTML input field was changed
An HTML button was clicked

Often, when events happen, you may want to do something.
