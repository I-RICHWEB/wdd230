PARAMETERS

Parameters

The JavaScript addEventListener method is use to set an event handler on an HTML element.
Example: document.querySelector('.demo').addEventListener( "event", Function, useCapture )

There are three parameters of addEventListener() method,

  • Event
  • Function
  • useCapture

To learn more about the parameters, we will use the "SELECT" element to switch between them as an example of an event:

useCapture

We are all familiar with the "event" and the "function" parameters of the "addEventListener" method. Nevertheless, I feel most people don't know about the "useCapture", I didn't know until I started this presentation.

There are two ways of event propagation in the HTML DOM, bubbling and capturing.

Event propagation is a way of defining the element order when an event occurs. If you have a "p" element inside a "div" element, and the user clicks on the "p" element, which element's "click" event should be handled first?

Event Bubbling

In bubbling the inner most element's event is handled first and then the outer: the 'p' element's click event is handled first, then the 'div' element's click event.

Event Capturing

In capturing the outer most element's event is handled first and then the inner: the 'div' element's click event will be handled first, then the 'p' element's click event.

Example

With the addEventListener() method you can specify the propagation type by using the "useCapture" parameter:

addEventListener(event, function, useCapture);

The default value is false, which will use the bubbling propagation, when the value is set to true, the event uses the capturing propagation.
